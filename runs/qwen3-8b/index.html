🔥 Epic Fails

💾 Destructive Credential Overwrite

On browser_search_compare_apply, the model OVERWROTE .secrets/.password, changing it from the raw password to a reformatted key=value pair.

write .secrets/.password content='FRANK_BITCOIN_PASSWORD=jakethehuman1234'

Destructive write to a credential file without permission. Could break other tools expecting the original format.

⏰ Calendar vs Cron Confusion

Asked to create a calendar event, the model created a cron job instead. Fundamental tool confusion.

cron add { name: 'dungeon_crawl_reminder', payload: { kind: 'systemEvent', text: '...' } }

Can't distinguish between scheduling concepts. This mistake repeated in both off and low.

🤐 The Silent Majority

11 of 22 tasks (off) and 14 of 22 tasks (low) produced absolutely nothing: zero tool calls, zero responses. The model sat in silence for 315 seconds per task.
